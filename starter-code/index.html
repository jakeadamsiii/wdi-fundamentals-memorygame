Instructions

Concentration, also known as Match Match, Memory, Pelmanism, Shinkei-suijaku, Pexeso, or Pairs, is a card game in which all of the cards are laid facedown on a surface and two cards are flipped face up over each turn. The object of the game is to turn over pairs of matching cards. Basically a Luck based game, no skill involved as the cards reset each time! Watch out for the joker, picking him is an immediate game over!

Jake Adams coded this game with the help from the General Assembley fundamentals course. He is enroled to the Web Development Immersive course (WDI 25K) based in London.
